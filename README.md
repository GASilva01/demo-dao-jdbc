# demo-dao-jdbc

Projeto desenvolvido durante o módulo de acesso a banco de dados do curso "Java COMPLETO Programação Orientada a Objetos" do professor Nélio Alves.

Neste projeto foi implementado um CRUD completo utilizando o padrão DAO (Data Access Object) para manipulação de dados com MySQL e JDBC puro, sem frameworks.

**`Tecnologias`**

* #### Java 25 (Zulu LTS)

* #### MySQL JDBC

**`Como configurar`**

* Clone o repositório

* Copie db.properties.example para db.properties

* Preencha com suas credenciais do MySQL

* Execute a classe Main
